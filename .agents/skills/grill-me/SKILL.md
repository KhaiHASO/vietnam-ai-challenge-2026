# Skill: Grill Me

## Purpose
Interrogate the developer's design goals and requirements one question at a time before any code or specifications are written.

## Rules
1. Ask one clarifying question at a time.
2. Do not write any code or draft any specs until the interrogation is finished.
3. Stop and wait for user feedback.
4. Dive into edge cases, technical trade-offs, and fallback strategies.
5. Synthesize the answers into the final `proposal.md`.
